# HDI Predictor using Machine Learning

## Project Overview
The HDI Predictor is a Machine Learning and Flask-based web application that predicts the Human Development Index (HDI) of a country based on key socio-economic indicators.

## Problem Statement
Predict the Human Development Index (HDI) using Machine Learning by analyzing:
- Life Expectancy at Birth
- Expected Years of Schooling
- Mean Years of Schooling
- Gross National Income Per Capita

## Features
- HDI Prediction using Machine Learning
- User-friendly Flask Web Interface
- Real-time Prediction
- Simple and Responsive UI
- Fast Prediction Results

## Installation

```bash
git clone https://github.com/ramyamamidisetti/HDI_Predictor.git
cd HDI_Predictor
pip install -r requirements.txt
python app.py
```

## Output
1. Open the application in your browser.
2. Enter:
   - Life Expectancy
   - Expected Years of Schooling
   - Mean Years of Schooling
   - Gross National Income Per Capita
3. Click **Predict HDI**.
4. The predicted HDI value will be displayed.
