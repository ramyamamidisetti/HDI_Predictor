- HTML
- CSS
- Joblib

## Description

This project predicts the Human Development Index (HDI) using Machine Learning. The prediction is based on four input parameters:

- Life Expectancy at Birth
- Expected Years of Schooling
- Mean Years of Schooling
- Gross National Income (GNI) Per Capita

The application uses a trained Machine Learning model and a Flask web application to generate accurate HDI predictions through a simple user interface.
